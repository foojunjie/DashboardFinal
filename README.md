# DashboardFinal
DashboardFinal is a dashboard that shows the Monthly Outper per Workcell and the live data of Stock In and Out of Product E24-0100-WF and E25-0100-WF.

# Download and Installation
To start running the dashboard, the installation of the following steps are needed.
* [Download the ODBC driver 17 to access the Maps database and run it](https://learn.microsoft.com/en-us/sql/connect/odbc/download-odbc-driver-for-sql-server?view=sql-server-ver17)
* Download the code.zip
* Follow the Installation.txt
